# Shopping Cart

## Shopping Cart - Overview

This is the third project of the "React" course within "The Odin Project".

This project solidifies the core principles of React.

In particular, this project aims to create an online store application.

**You can preview the project at: https://dantesecommerce.netlify.app/**

Note, this project utilizes some "Ant Design" components.

## Skills Demonstrated

- React

- Client-Side Routing

- Single-Page Applications

- React Router

- Data fetching with React

- Testing Library

- Vitest
